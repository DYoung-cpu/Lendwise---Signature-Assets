# LendWise Mortgage - Email Signature Assets

This repository hosts all image assets used in LendWise Mortgage email signatures.

## Purpose

Provides reliable, company-controlled hosting for email signature images to ensure:
- ✅ High availability (no third-party dependencies)
- ✅ Version control for all visual assets
- ✅ Professional brand consistency
- ✅ Zero risk of images breaking

## Directory Structure

```
signature-assets/
├── logos/          # Company logos and variations
├── buttons/        # Call-to-action buttons (Apply Now, etc.)
├── icons/          # Legal/compliance icons (Equal Housing, etc.)
└── README.md       # This file
```

## Logos (`/logos`)

- `owl-logo.png` - LendWise owl logo (default)
- `owl-gif.gif` - Animated owl logo
- `circle-text-logo.png` - LendWise text in circle
- `green-background-logo.png` - Logo with green background
- `circle-background-text.png` - Text logo with circle background
- `headshot-placeholder.png` - Employee headshot placeholder
- `static-template-owl.gif` - Static template owl GIF
- `owl-over-text-vertical.png` - Owl positioned over text (vertical)
- `owl-text-horizontal.png` - Owl and text horizontal layout
- `green-background-outlook.png` - Green background for Outlook compatibility

## Buttons (`/buttons`)

- `apply-now-button.png` - Main call-to-action button for loan applications

## Icons (`/icons`)

- `equal-housing-logo.png` - Equal Housing Opportunity logo (required for compliance)

## Usage

Images are accessed via GitHub raw URLs in the email signature generator:

```
https://raw.githubusercontent.com/[username]/lendwise-signature-assets/main/[folder]/[filename]
```

Example:
```
https://raw.githubusercontent.com/[username]/lendwise-signature-assets/main/buttons/apply-now-button.png
```

## Maintenance

- All images are migrated from Imgur for reliability
- Original Imgur URLs documented in signature generator code
- Update this repo for any new signature assets
